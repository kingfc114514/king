\# ✈️ 飞机大战 · 控制台射击游戏



一款基于 C# 控制台开发的经典射击游戏，玩家操控战机在固定区域内移动，击落敌机并挑战 BOSS，体验逐渐提升的难度与成长系统。



\---



\## 🎮 游戏简介



\- \*\*玩家控制\*\*：战机在屏幕左侧区域移动，发射子弹攻击敌机。

\- \*\*敌人生成\*\*：普通敌机周期性出现，BOSS 在每轮清理后登场。

\- \*\*数值成长\*\*：通过击落敌机获得分数，积攒奖励点可提升攻击力或生命值。

\- \*\*动态难度\*\*：敌机与 BOSS 属性随关卡等级提升，挑战逐步升级。



\---



\## 🕹️ 操作说明



| 按键 | 功能           |

|------|----------------|

| W/A/S/D | 上下左右移动 |

| J | 攻击（发射子弹） |

| 1 | 消耗 1 点奖励，提升 1 点攻击力 |

| 2 | 消耗 1 点奖励，恢复生命值（回复量随等级增加） |



> 注：奖励点通过击败 BOSS 获得，上限可无限累积。

\## 🔧 运行方法



\### 环境要求

\- .NET Framework 4.7.2 或 .NET Core 3.1 / .NET 5+ （支持控制台程序）



\### 启动步骤

1\. 克隆或下载本项目源码。

2\. 使用 Visual Studio 或 VS Code 打开项目文件夹。

3\. 确保 `Program.cs` 为启动项，按 `F5` 或 `dotnet run` 运行。

4\. 游戏窗口大小已预设为 80×50，若控制台缩放异常，请手动调整字体大小。

\## ✨ 技术亮点



\- \*\*纯控制台图形绘制\*\*  

&#x20; 使用 `Console.SetCursorPosition` 与 ANSI 颜色实现动态帧绘制，无需第三方库。



\- \*\*面向对象架构\*\*  

&#x20; 抽象类、接口、继承与多态构建灵活的游戏对象体系，易于扩展新敌人类型。



\- \*\*场景状态机\*\*  

&#x20; 通过 `ISceneUpdate` 接口管理开始、游戏、结束三场景，切换流畅。



\- \*\*碰撞检测\*\*  

&#x20; 基于像素级坐标匹配实现精准碰撞，支持玩家、敌机、子弹之间的交互。



\- \*\*对象池优化\*\*  

&#x20; 子弹数组复用，减少频繁创建销毁带来的 GC 开销。



\- \*\*实时输入响应\*\*  

&#x20; 非阻塞键盘监听（`Console.KeyAvailable`），保证游戏帧率稳定。
